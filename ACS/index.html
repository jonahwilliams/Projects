Urban/Suburban Poverty Analysis using ACS data

BGD on a logit model - predict percent of population in poverty given other percentage variables for census block groups in Iowa and Nebraska. This is an exporatory project that aims to identify links between poverty in urban and suburban enviornments, wtih respect to their differences and neighborhoods which are difficult to model

def logit(data, parameters, bias):
    return 1 / (1 + np.exp(-(np.dot(data, parameters) + bias)))

y = suburban[["pct_Prs_Blw_Pov_Lev_ACS_08_12"]].values
X = suburban[[col for col in suburban.columns if col != "pct_Prs_Blw_Pov_Lev_ACS_08_12"]].values

sub_parameters = []
sub_biases = []
sub_errors = []
for i in range(100):
    error = []
    param = np.random.normal(0, 0.01, (len(X[0])))
    bias = 0
    for i in xrange(1000):
        if (i % 100 == 0):
            error.append(np.mean((logit(X, param, bias) - y[0::, 0])**2))
        param += -0.01 * np.dot((logit(X, param, bias) - y[0::, 0]) * (1 - logit(X, param, bias) * logit(X, param, bias)), X)
        bias += -0.01 * np.mean((logit(X, param, bias) - y[0::, 0]) * (1 - logit(X, param, bias) * logit(X, param, bias)))
    sub_errors.append(error)
    sub_parameters.append(param)
    sub_biases.append(bias)

Residual Analysis

Because the model was chosen and fit on all of the data, we can not get an unbiased estimate of out of sample error. This was outside the scope of the project however, especially due to the small amount of data. Instead we will use the optimal parameter estimates and look for local co-occurance of high residual values, essentailly identifying neighborhoods with undocumented but significant local explanatory variables.

def calc_mode(x):
    '''Approximates the mode of the data by resampling and using numpy.mode'''
    from scipy.stats.mstats import mode
    return mode(np.random.choice(x, 100000))[0]

def closest_breakpoint(x, std):
    if (x > 0 - std) and (x < 0 + std):
        return 3
    elif (x >= 0 + std) and (x < 0 + 2*std):
        return 4
    elif (x >= 0 + 2*std) and (x < 0 + 3*std):
        return 5
    elif (x >= 0 + 3*std):
        return 6
    elif (x <= 0 - std) and (x > 0 - 2*std):
        return 2
    elif (x <= 0 - 2*std)and (x > 0 - 3*std):
        return 1
    else:
        return 0
